# SR-RAG: Data Release

This repository contains the benchmark and knowledge graph for the paper:
*Integrating Evidence-Based Medicine into GraphRAG*.

## Contents

- `benchmark/`: SR-RAG benchmark (1,637 QA pairs)
- `knowledge-graph/`: SR-RAG knowledge graph (357,844 nodes, 371,226 edges)

See each subdirectory for detailed schema documentation.
